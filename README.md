
# Titanic Survival Prediction 🚢

This is my first Machine Learning model training project. In this project, I worked with the Titanic dataset and learned the basic process of data preprocessing, model training, prediction, and accuracy comparison.

## What I Did

- Loaded and explored the Titanic dataset
- Handled missing values in the dataset
- Converted categorical data into numerical form using Label Encoding
- Split the dataset into training and testing data
- Trained different Machine Learning classification models:
  - Logistic Regression
  - K-Nearest Neighbors (KNN)
  - Naive Bayes
- Compared the accuracy of all three models

## Model Performance

With `test_size = 0.2`:

- Logistic Regression: ~80%
- KNN: ~70%
- Naive Bayes: ~77%

Among these models, Logistic Regression gave the best accuracy in my experiment.
